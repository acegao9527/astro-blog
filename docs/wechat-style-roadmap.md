# 公众号极简样式优化路线图

> 目标：把当前博客逐步打磨成接近优质微信公众号技术长文的阅读体验，风格关键词为 **极简、克制、学术感、技术研究风**。

## 优化背景

参考样式来自一篇排版非常克制的微信公众号技术文章。它的核心特点不是装饰多，而是：

- 正文干净，深灰字色，行距舒适
- 标题克制，主要依靠字号、字重和细分隔线建立层级
- 技术块自然融入正文，不做厚重卡片
- 通篇视觉语言统一，几乎不依赖彩色强调

这类风格非常适合：
- 技术分析
- 安全研究
- 系统拆解
- 深度教程
- 工具评测

## 目标风格定义

### 风格关键词
- 极简
- 克制
- 学术感
- 技术长文
- 安全研究风

### 预期效果
让博客文章页具备以下感受：
- 第一眼显得轻而不空
- 读长文不累
- 标题层级清晰但不夸张
- 代码、引用、图表与正文自然融合
- 更像研究报告，而不是营销卡片页

## 样式拆解结论

### 1. 正文系统
建议目标参数：
- 正文字号：`16px`
- 行高：`1.8`
- 文字颜色：`#333333`
- 字重：`400`
- 段间距：`0.8em 0`

### 2. 标题系统
#### 一级小节标题
建议目标参数：
- 字号：`26px ~ 28px`
- 字重：`700`
- 标题颜色：`#222222`
- 下边框：`1px solid #EAEAEA`
- 上边距：`24px`
- 下边距：`16px`
- 行高：`1.2`

#### 次级标题
建议目标参数：
- 字号：`20px`
- 字重：`700`
- 颜色：`#333333`
- 不加背景色

### 3. 代码块
建议目标参数：
- 字号：`14px`
- 等宽字体
- 背景：`#F7F7F7`
- 边框：`1px solid #EEEEEE`
- 避免黑底和过重阴影

### 4. 配色系统
建议目标参数：
- 主文字：`#333333`
- 标题：`#222222`
- 辅助文字：`#666666`
- 分隔线：`#EAEAEA`
- 代码背景：`#F7F7F7`
- 页面背景：`#FFFFFF`

## 实施路线图

### 阶段 1：正文基础样式统一
目标：先把读起来舒服这件事做好。

任务：
- 调整文章页正文的字号、行高、段间距
- 统一正文文字颜色和字重
- 检查移动端和桌面端阅读密度
- 去掉过重、冲突或噪音型样式

验收标准：
- 长文首屏更通透
- 连续阅读 5 到 10 段不会明显累
- 手机端不显得拥挤

### 阶段 2：标题层级重构
目标：建立像公众号技术长文那样稳定、克制的标题层次。

任务：
- 重做 `h2` / `h3` 样式
- 给主小节标题增加细分隔线
- 调整标题上下留白
- 控制标题与正文的视觉过渡

验收标准：
- 小节标题一眼可扫
- 层级清晰但不过度强调
- 通篇标题节奏统一

### 阶段 3：技术内容组件轻量化
目标：让代码、引用、表格、图片说明都更自然。

任务：
- 重做代码块样式，降低视觉重量
- 优化引用块样式，避免大面积深色背景
- 优化图片说明文字与图文间距
- 检查表格在窄屏下的可读性

验收标准：
- 代码块像研究文插图，而不是 UI 组件
- 引用块不喧宾夺主
- 图表与正文衔接顺畅

### 阶段 4：文章模板与写作约束
目标：把样式优势转化成稳定产出能力。

任务：
- 固化一套技术文章结构模板
- 定义 Markdown 层的标题、段落、引用、代码使用规范
- 尽量让作者不依赖手工排版也能写出接近目标风格的文章

建议模板：
- 引言
- 问题是什么
- 我们如何发现
- 核心分析
- 关键证据
- 影响与判断
- 修复方案
- 总结

### 阶段 5：博客与公众号双端统一
目标：让博客风格与公众号输出形成统一品牌感。

任务：
- 提炼博客样式 token
- 反推一套公众号排版规范
- 建立博客正文到微信技术长文的映射关系

## 建议的执行顺序

建议按以下优先级推进：
1. 正文基础样式统一
2. 标题层级重构
3. 代码块 / 引用块轻量化
4. 写作模板与 Markdown 约束
5. 博客 / 公众号双端统一

## 实现建议（工程层）

建议重点排查和调整的文件：
- `src/styles/global.css`
- `src/layouts/Layout.astro`
- `src/pages/post/[slug].astro`

建议把文章正文样式尽量收敛在统一选择器中，而不是散落在多个局部组件内。这样后面调整整体风格时成本更低。

## 不建议的方向

为了保持这套气质，不建议：
- 使用彩色标题底条
- 使用渐变背景
- 使用重阴影卡片
- 使用大面积高亮色强调
- 为每个区块加复杂装饰
- 过多使用居中排版

> 越像研究报告，越接近这套样式的高级感。
